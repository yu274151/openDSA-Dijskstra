Welcome to the tutorial for the DEFORMS problem solving system!

We'll guide you through the basic steps on how to use this system to solve equations, submit answers, and get feedback. Important: Please use the Google Chrome browser to use this system for the best experience. Otherwise, the user interface may not appear correctly.

1. Getting started: The workspace

The workspace is the white area marked with "wkN" on the left side. This is where one adds all the equations from the equation bank to create a system of equations to solve. One can add more workspaces to create separate areas for trying out different variations of a problem, or delete entire workspaces to create a clean slate.

2. Adding equations and populating with values

To add an equation to a workspace, select the proper page of equations from the drop down menu on the right, and click on an equation. When it is highlighted, it is ready to be added.
Then, click on the Add button for the workspace you want to add it to. A boxed version of the equation will appear. You can click on the boxes to bring up different options.
To add a value from the question prose, click on the value box to highlight it,
and then click on the box in the equation to place it in.
Units of quantities can be changed by clicking on the unit part of the populated values to bring up a list of units to convert to.
You can create simple arithmetic equations from the Arithmetic equations palette by adding values and associating variables, as outlined later.

3. Associating variables (for multivariable systems)

To create a system of equation with more than one variables, where the same variable appears in another equation, one needs to associate variables. This is done by clicking on an empty box in an equation, clicking on "start association" in the menu to highlight the box, then clicking on another empty box to associate it with. 4. Solving equations and submitting answers

To solve a system of equations, after you have populated the values and associated variables, click the check mark on the left side to select (☑) the equation(s), and click on the "Solve button" for that workspace.

This will create a set of boxes with solutions for the system you created.
Click on this box once to highlight it, and then click on the yellow submission box for the question you are answering to submit this answer.
Some cases, you may have to click on a box to select a value from a menu to input. In both cases, the scissors can be clicked to clear the box.

5. Receiving feedback

The notifications panel on the right side shows errors for anything that may have happened with your system. Click on the errors to show more details about them. In some cases, clicking on the boxes will also show where in the equations an error has occurred.

6. Additional help
